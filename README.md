# traffic-accident-analysis-ml
Traffic Accident Analysis &amp; Severity Prediction using Machine Learning
## Project Overview
This project analyzes a dataset of over 1.5 million UK traffic accidents and applies machine learning to predict accident severity.

## Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Machine Learning (Random Forest)
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn

## Results
The model was able to identify key factors influencing accident severity such as time of day and road conditions.
